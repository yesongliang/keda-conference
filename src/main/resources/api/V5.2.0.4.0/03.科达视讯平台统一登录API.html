名称 token获取
URI /api/v1/system/token
方法 POST
说明 token获取
最低支持版本 1

参数说明

参数名称 数据类型 说明
oauth_consumer_key* str oauth中认证的软件key
oauth_consumer_secret* str oauth中认证软件key对应的value

成功回复示例

{
  "success": 1,
  "account_token": "1607660e3ba14f26db618372f4c60405"
}

成功回复说明

参数名称 数据类型 说明
success int 是否成功
account_token str token，应用权限

失败回复示例

{
  "success": 0,
  "error_code": 123456
}

失败回复说明

参数名称 数据类型 说明
success int 是否成功
error_code int 错误码
